# SARVAM POS - Quick Start Guide

## 🚀 Getting Started

### Testing the Application

1. **Start a New Bill**
   - Click "New Billing" on the home screen
   - Enter customer details (e.g., "John Doe", "9876543210")
   - Click "Start Billing"

2. **Add Products** 
   - Click "Scan Item" button
   - If camera permission is denied, the app automatically switches to **Manual Mode**
   - Enter one of these barcodes:
     - `8901234567890` - Organic Honey (₹450)
     - `8901234567891` - Basmati Rice (₹850)
     - `8901234567892` - Special Coffee (₹650)

3. **Complete Payment**
   - Click "Proceed to Payment"
   - Try **Cash Mode**: Enter 1000, see change calculated
   - Try **UPI Mode**: View mock QR code

4. **View Dashboard**
   - Go back to home screen
   - Click "Seller Login"
   - View sales analytics and charts

## 🎤 Voice Features

The AI voice assistant will:
- Welcome customers by name
- Announce each product when scanned
- State the final total

*Note: Click anywhere on the page first to enable voice (browser requirement)*

## 📷 Camera Issues?

**No worries!** The app handles this gracefully:

- **Permission Denied** → Automatically switches to Manual Mode
- **No Camera Found** → Shows manual entry with sample barcodes
- **Camera In Use** → Fallback to manual entry

You can always toggle between Camera and Manual modes using the toggle button.
